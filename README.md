### 中文名：简单跑跑

### English Name: Hello Runner 

**一款简单直接的跑步应用.**

**(大家都很忙，省点时间去玩虚的，还是直入主题抓重点！)**

计划的功能:

+ 耐力跑：计时、计步 => 配速 距离 燃脂

   （设置一个距离，开始跑，预计到距离后提醒并指导转为放松模式 ）

   （设置一个时长，开始跑，预计到距离后提醒并指导转为放松模式 ）

    设定配速，高于或者低于预设配速10%，则进行提醒
 
+ 变速跑：法特莱克跑 （1分钟间隔的快慢交替跑）

+ 跑步指导（一些跑步知识文库）

+ 跑步日志

+ 目标训练（5km / 10km / 半马 / 全马 ）

[其他待定功能: 朋友圈分享 ]
